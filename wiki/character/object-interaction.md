# 캐릭터 - 사물 상호작용

---

## 작동체계

### 상호작용 인터페이스

**가이드라인 표시**
- 작동 가능한 사물의 가이드 범위(점선) 내로 접근 시, 대상 물체에 테두리가 표시되어 상호작용 가능 물체임을 표시

**상호작용 표시**
- 가이드 범위 + 시선이 대상을 향할 때 상호작용 버튼이 표시됨
- 모션 지정 시 상호작용 준비 모션으로 변경
- 상호작용 버튼을 누르면 상호작용 시작

**다중 상호작용**
- 여러 상호작용이 가능한 경우, 리스트가 버튼과 함께 표시
- 단축키: F-G-H-J (최대 4종)
- 상호작용 안내 중 해당 키가 최우선 할당
- 게임패드: 상호작용 키 길게 누르고 + 십자키로 선택 후 손 떼면 실행

## 사물 상호작용

### 문

- 문의 종류에 따른 전용 모션 사용
- 별도 모션 미지정 시 동작 없이 열기/닫기 가능

| 문 종류 | 설명 |
|---------|------|
| 여닫이문 | 경첩 방향에 따라 밀거나 당겨서 열고 닫음. 열리는 방향 고정 |
| 미닫이문 | 슬라이드로 열고 닫음. 벽에 막힌 반대편에서는 열 수는 있으나 닫을 수 없음 |
| 컷씬 문 | 상호작용 모션이 짧은 컷씬처럼 구성. 도중 다른 행동 불가. 중요도 높은 문 연출용 |

### 스위치

- 문과 마찬가지로 종류에 따른 전용 모션 사용
- 모션 미지정 시 동작 없이 작동

| 스위치 종류 | 설명 |
|------------|------|
| 버튼 스위치 | 누를 때마다 작동. 상태 변경 없음 |
| 전환 스위치 | On/Off 상태에 따라 형태 변경. 상호작용 시 반대 상태로 전환 |
| 컷씬 스위치 | 컷씬 문과 동일. 도중 다른 행동 불가 |

### 살펴보기

- 문/스위치 외 환경 루팅 등을 위한 환경 상호작용
- 살펴보는 동안 지정 자세 유지, 종료 시 자세 해제

| 단계 | 설명 |
|------|------|
| 살펴보기 시작 | 전용 시작 모션. 컷씬 모션 사용 시 다른 행동 차단 |
| 살펴보기 지속 | 루프 모션 (정지/뒤지기/옮기기 등). 루팅의 경우 루팅 인벤토리 표시 |
| 살펴보기 종료 | 마무리 모션 사용 |
