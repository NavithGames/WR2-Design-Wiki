# 캐릭터 - 지형 상호작용

---

## 지형 상호작용 정의

- 일반적인 은폐/엄폐는 지형 상호작용이 아님
- 지정된 장소에서 약속된 액션을 취했을 때 일어나는 상호작용 (주로 지형 이용)
- 지형에 미리 약속된 상호작용 트리거를 배치 → 트리거 접촉 상태에서 액션 실행 시 상호작용 시작

---

## 지형 상호작용 종류

### 등반 (Climb)

- 점프로 넘을 수 없지만 손동작으로 타오를 수 있는 높이의 지형을 올라타는 동작
- 가슴 높이 ~ 최대 ___cm
- 불특정 공간에서의 등반 방지를 위해 **지정 장소에만 등반 볼륨 설치**
- 해당 위치에서 전용 인터렉션 버튼으로만 등반 가능

| 시점 | 모션 |
|------|------|
| 1인칭 | 무기를 집어넣고 양팔을 눈앞으로 뻗었다가 가슴으로 쓸어내리며 올라타기 |
| 3인칭 | 벽 오르기 (뛰어넘지 않고 올라서기) |

### 타넘기 (Vault)

- 비교적 낮은 장애물을 반대편까지 뛰어넘는 동작
- 점프보다 신속하고 시원한 느낌 (파쿠르 스타일)
- 등반 = 높은 곳으로 올라서기 / 타넘기 = 평지 장애물 뛰어넘기

| 시점 | 모션 |
|------|------|
| 1인칭 | 왼손을 뻗어 장애물을 짚고 카메라워크로 타넘기 연출 |
| 3인칭 | 왼손으로 짚고 몸을 날려 타넘는 동작 |

### 사다리 매달리기

- **수직 사다리 잡기**: 평지나 사다리 중간에서 바로 매달리기 가능
- **사다리 위쪽 끝**: 몸을 아래로 내려 매달리기 동작으로 이어짐
- **Middle of Ladder Volume**: 사다리를 붙잡기
- **Top of Ladder Volume**: 올라타기 / 상단에서 매달리기로 전환
- 모든 사다리의 1단은 **30cm 단위로 통일** 필요

| 시점 | 모션 |
|------|------|
| 1인칭 | 사다리 타고 오르거나 내려가기 |
| 3인칭 | 사다리 타고 오르거나 내려가기 |

> 하나의 모션을 정방향 재생 / 정지 / 역재생으로 오르내림 표현

### 미끄럼틀 (Slide Slope)

- 상단에서 인터렉션 시 하단까지 슬라이드로 미끄러져 내려옴
- 경사각은 모두 **45도로 통일**
- 하단에서 상단으로는 인터렉션 불가

| 시점 | 모션 |
|------|------|
| 1인칭 | 별도 모션 없이 자동 이동 (사격 및 조준 가능) |
| 3인칭 | 하체만 주저앉은 모습으로 표현 |
