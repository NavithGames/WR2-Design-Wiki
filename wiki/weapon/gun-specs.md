# 총기 기본 스펙

총기별 기본 스펙 항목 정의. 각 총기의 데이터 시트에 들어가는 필드와 단위를 정의한다.

---

## 식별 정보

| 항목 | 단위 | 설명 |
|------|------|------|
| 코드네임 | 영문+숫자 | 총기 식별 코드번호 (M-16, AK-47 등) |
| name | 영문 | 총기 고유이름 (vector, scorpion 등 별도 닉네임) |
| 계열 | 타입 | HG=권총 / AR=돌격소총 / SMG=기관단총 / SR=저격소총 / SG=산탄총 / LMG=경기관총 |
| 탄종 | 타입 | 사용하는 탄종. LMG은 돌격소총~저격소총탄 겸용 (전용 탄종 없음) |
| 구경 | 타입 | 사용하는 탄종의 구경 (탄종별 특성 참고) |

---

## 탄창

| 항목 | 단위 | 설명 |
|------|------|------|
| 탄창 타입 | 타입 | 구경 및 형식에 따라 총기마다 사용 가능한 탄창 타입이 결정됨 |

---

## 무게 (부품별)

| 항목 | 단위 | 설명 |
|------|------|------|
| 리시버 무게 | kg | 기본 총몸 무게. 총기 무게 = 모든 기본 부품 무게 합산 |
| 개머리판 무게 | kg | 일체형이거나 없으면 0 |
| 총열 무게 | kg | 기본 총열 무게 |
| 총열 덮개 무게 | kg | 리시버 일체형이거나 없으면 0 |
| 권총손잡이 무게 | kg | 리시버 일체형이면 0 |

---

## 리시버 타입

| 타입 | 설명 |
|------|------|
| 1 | 일반적인 AR 형태. 개머리판 + 총열 + 권총손잡이 부품 구성 |
| 2 | 권총 형태. 개머리판은 어태치먼트, 권총손잡이 일체형, 총열 덮개 리시버 일체형 |
| 3 | 총열 덮개가 리시버와 일체형 (예: MP7) |
| 4 | 총열 덮개와 개머리판이 일체형 (예: M-1 개런드) |

---

## 부품 호환 타입

| 항목 | 단위 | 설명 |
|------|------|------|
| 총열 타입 | 타입 | 교체할 수 있는 총열 타입 |
| 개머리판 타입 | 타입 | 교체할 수 있는 개머리판 타입 |
| 권총손잡이 타입 | 타입 | 교체할 수 있는 권총손잡이 타입 |
| 상부레일 타입 | 타입 | 리시버에 장착할 수 있는 상부레일 어태치먼트 타입 |

---

## 속도 보정값

| 항목 | 단위 | 설명 |
|------|------|------|
| 기본 이동 속도 보정 | % | 무게와 관계없이 이동속도에 영향. 0.9 = 기본 속도의 90% |
| 조준 속도 보정 | % | 조준 모션의 기본 속도 배율. 튜닝/어태치먼트/캐릭터 특성 합산 보정값을 애니메이션 길이에 곱하는 방식 |
| 비 조준 속도 보정 | % | 조준→지향사격 전환 애니메이션의 기본 속도 보정값 |
