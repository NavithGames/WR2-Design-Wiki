# 총기 커스터마이징

---

## 기조

- 필요한 어태치먼트만 채용
- 기본 총기 구성을 변경하는 부품은 **튜닝**에서 관리 (개머리판, 리시버, 총열 등)
- 어태치먼트나 튜닝부품은 가방/파우치에 보관 가능
- 탄창은 개별 아이템으로 탄입대에 장착하지만 사용중인 총기에 튜닝 효과로 적용될 수 있음
- 리로드는 다음 탄입대에 있는 탄창으로 교체하는 것 (현재 탄창에 탄을 채워넣는 것이 아님) *(결정 필요)*

---

## 어태치먼트 구조

### 총몸 소켓

소총 기준으로 리시버를 중심으로 스톡, 그립, 총열로 구성된 총몸 구성요소.

- 총몸은 튜닝을 장착하지 않아도 기본 부품이 장착되어 있으며, 튜닝으로 교체하면 교체 부품의 외형 및 성능으로 교체
- 교체 이전 부품은 기본 가방으로 이동
- 부품은 가방에 보관 가능 (조립된 총기는 불가)
- **리시버**가 스톡-총열-그립의 소켓을 가짐
  - 소켓이 없는 리시버로 교체 시 해당 소켓의 바디는 장착 해제 → 인벤토리 이동
  - 총열과 그립 소켓은 모든 리시버가 반드시 보유
  - 일부 리시버만 스톡 소켓 미보유
- 총열 덮개에 사이드레일/언더레일이 없으면 이전 어태치먼트는 해제 → 가방 이동
- 총열 교체 시 총열 덮개가 호환되지 않으면 해제 → 가방 이동

### 총몸 부품 역할

| 부품 | 역할 |
|------|------|
| **리시버** | 총기의 중심 부품. 각 부품의 장착 기초. 연사속도, 총구속도 등 사격 특성 결정 |
| **개머리판** | 견착 시 성능 관련 (일부 총기에는 없음) |
| **권총손잡이** | 전반적인 안정감 관련 |
| **총열** | 사격 정밀도와 탄도 특성에 영향 |
| **총열 덮개** | 총열 보호. 미장착 시 비조준 사격만 가능. 사격 안정성, 사이드레일/언더레일 제공 |

---

## 어태치먼트

총몸 구성요소에 연결된 부착물 형태의 소켓.

### 리시버에 장착
- 가늠자 및 스코프
- 액세서리 (장식물)

### 총열 덮개에 장착
- 사이드레일
- 언더레일

### 총열에 장착
- 가늠쇠
- 머즐

> 바디에 따라 소켓이 없는 경우 해당 어태치먼트는 인벤토리로 자동 이동

### 가늠자/가늠쇠 고정 총기
- M-4: 가늠자 교체 가능, 가늠쇠 교체 불가
- K-2: 가늠자/가늠쇠 모두 교체 불가
- 리시버에 레일이 없는 총기 (K-2 등): 스코프 장착 시 레일 부품이 덧붙여지는 형태

---

## 탄창

탄창은 기본 부품이 아니라 사격을 위해 필수적으로 사용해야 하는 탄약 급탄장치로, 같은 탄 구경이면 총기 종류가 달라도 공통 형식의 탄창은 상호 호환됨.

- **옵션 탄창**으로 교체 시 해당 탄창 사용 동안 옵션 효과 획득
- **삽탄**: 탄창은 탄입대에 장착된 순서로 차례대로 사용. 빈 탄창은 삽탄으로 채울 수 있음 (인벤토리에 탄환 필요). 한 번에 하나씩 자동 삽입, 상대적으로 긴 시간 필요
- **리로드**: 총기에 삽입된 탄창을 다음 순서의 탄입대 탄창으로 교체. 빈 탄창도 버리지 않음. 리로드를 자주 하면 덜 채워진 탄창이 많아져 추후 오랜 삽탄 필요

---

## 총기 계열별 어태치먼트 구성

### 부품 호환성 기조
- 모든 부품은 계열별 총기 전용 (일부 스코프만 상부 레일에 공용 장착 가능)
- 돌격소총 스코프는 기관단총, 저격, 경기관총에도 병용 가능

---

### 권총

**특징**: 총몸에 개머리판 슬롯 없음 (어태치먼트로 장착). 리시버에 언더레일 장착. 전용 스코프 사용.

| 슬롯 | 어태치먼트 |
|------|-----------|
| 메인 스코프 | 가늠자(가늠쇠), 닷 사이트, 홀로그램 사이트, 저배율 스코프 |
| 총구 | 소음기, 컴펜세이터, 소염기 |
| 언더레일 | 플래시라이트, 레이저 사이트, 복합 사이트 |
| 탄창 | 폴리머 탄창, 확장 탄창, 테프론 탄창 |

---

### 돌격소총

**특징**: 기본 부품 + 어태치먼트 특성 동일.

| 슬롯 | 어태치먼트 |
|------|-----------|
| 메인 스코프 | 가늠자, 닷 사이트, 저배율 스코프, 고배율 스코프, 복합 스코프 |
| 총구 | 소음기, 소염기 |
| 탄창 | 폴리머/확장/테프론 탄창 |
| 언더 총열 | 수직 손잡이, 앵글 그립 |
| 측면 총열 | 레이저사이트, 플래시라이트, 복합 사이트 |

---

### 저격소총

**특징**: 저격 소총 전용 스코프 장착 가능.

| 슬롯 | 어태치먼트 |
|------|-----------|
| 메인 스코프 | 가늠자 가늠쇠, 저배율 스코프, 고배율 스코프 (기본) |
| 탄창 | 폴리머/확장/테프론 탄창 |
| 총구 | 소음기 (소음 효과 제한적, 사거리 감소 큼), 소염기 |
| 언더 총열 | 삼각대 (특정 자세에서 안정화 향상) |

---

### 기관단총

**특징**: 총열 덮개 슬롯 없음. 언더레일/사이드레일은 리시버에 일체형.

| 슬롯 | 어태치먼트 |
|------|-----------|
| 메인 스코프 | 닷 사이트, 저배율 스코프 (사거리 증가) |
| 탄창 | -- |
| 총구 | 소음기, 소염기 |
| 측면 총열 | 플래시라이트, 레이저사이트 |

---

### 경기관총

**특징**: 전용 탄창 사용.

| 슬롯 | 어태치먼트 |
|------|-----------|
| 메인 스코프 | 가늠자, 닷 사이트, 저배율/고배율/복합 스코프 |
| 총구 | 소음기, 소염기 |
| 탄창 | 폴리머/확장/테프론 탄창, 박스 탄창 (일부 총기, 장탄량 30발로 감소 + 재장전 속도 빨라짐) |
| 언더 총열 | 수직 손잡이, 양각대 (특정 자세에서 집탄 범위 축소) |

---

### 산탄총

| 슬롯 | 어태치먼트 |
|------|-----------|
| 메인 스코프 | 가늠자 가늠쇠, 닷 사이트, 저배율/복합 스코프 |
| 총구 | 소음기, 소염기 |
| 탄창 (탄창 사용 총기) | 폴리머/확장/테프론 탄창. 확장 탄창은 드럼 형태 (12~20발) |
| 리로더 (튜브 탄창 총기) | 퀵 리로더 (한 번에 여러 발 장전) |
| 언더 총열 | 수직 손잡이, 플래시라이트, 레이저사이트, 복합사이트 |
