# Skins

---

## Skin Application Methods

### Texture Skins
- Material-swap only on shared models
- Concept of applying paint over the base weapon form

### Model Skins
- Uses dedicated models with dedicated materials
- Models must remain compatible with base animations

---

## Skin Production Philosophy

### Shared Base Animations
- Weapons sharing the same code name (M16, AK47, etc.) use the same base animations
- Applied by overriding only specific animations when a skin is equipped
- Special care needed to ensure hand shapes (grip form) do not change between skins

---

## Skin Effects

| Effect | Description |
|--------|-------------|
| **Appearance** | Purely visual differentiation |
| **Exclusive attachments and tuning** | Dedicated attachments/tuning only usable while the skin is equipped |
| **Exclusive animations** | Equip/unequip, reload, weapon inspection, emote animations |
