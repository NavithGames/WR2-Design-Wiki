# 총기 관련 모션 목록 (1인칭)

---

## 장비 및 조준

| 구분 | 모션명칭 | 비고 | 키 |
|------|----------|------|-----|
| 장비 | equip | 등에 메고 있던 총기를 꺼내 지향사격 준비 자세로 전환. 장전 모션 포함, 접철식 부품 펼치기 포함 | -- |
| 해제 | unequip | 사격 준비 자세에서 무기를 집어넣는 동작. 접철식 부품 동작 생략 (신속하게 등 뒤로) | -- |
| 조준 | aimin | 지향사격 자세에서 조준점으로 시점 정렬. 조준경에 의해 조준점 높이 변경 시 블렌딩 필요 | RMB |
| 조준해제 | aimout | 조준 자세에서 지향사격 자세로 변경 | RMB |

---

## 격발 및 장전

| 구분 | 모션명칭 | 비고 |
|------|----------|------|
| 격발 스타트 | fire_start | 지향사격 중 격발 시작 |
| 격발 루프 | fire_loop | 지향사격 중 격발 반복 |
| 격발 마무리 | fire_finish | 지향사격 중 격발 마무리 |
| 격발-조준 스타트 | aimfire_start | 조준중 격발 모션 |
| 격발 조준 루프 | aimfire_loop | 조준중 격발 반복 |
| 격발 조준 마무리 | aimfire_finish | 조준중 격발 마무리 |
| 볼트액션 | bolt_action | 격발 후 재장전 모션 (볼트액션 총기) |
| 탄창제거 | release magazine | 현재 탄창 제거. 총기 종류에 따라 노리쇠 후퇴고정 등 추가 모션 |
| 탄창삽입 | insert magazine | 새 탄창 삽입. 탄입대 순번에 맞는 외형 탄창 장착 |
| 탄창제거조준 | release magazine aim | 조준 중 왼손으로 탄창 제거 |
| 중간 재장전 | reload | 탄창 속 탄알이 남은 상태에서 장전 모션 |
| 중간 조준재장전 | aimreload | 조준 중 탄창 속 탄알이 남은 상태에서 장전 모션 |
| 재장전 | reloadempty | 탄창 속 탄알 다 소비 후 장전 모션 (노리쇠 전진, 펌프액션 등). **탄창이 모두 비기 전에는 별도 장전 불필요** |
| 조준 재장전 | aimreloadempty | 조준 중 탄창 속 탄알 다 소비 후 장전 모션 |

## 이동

| 구분 | 모션명칭 | 비고 |
|------|----------|------|
| 달리기 | run | 표준 이동. 지향사격 자세 유지한 채 달리기 |
| 조준달리기 | run aim | 조준 유지하면서 총기 약간 흔들림. 필요없음 |
| 걷기 | walk | 발소리 죽이고 천천히 걷기 |
| 조준 걷기 | walk aim | 조준중 발소리 죽이고 천천히 걷기. 필요없음 |
| 스프린트 시작 | spring start | 전력질주 시작, 총을 늘어뜨림. 필요없음 |
| 스프린트 루프 | sprint loop | 전력질주 반복 |
| 스프린트 끝 | sprint end | 전력질주에서 지향사격으로 복귀. 필요없음 |
| 쪼그려 이동 | crouch run | 앉은 상태에서 일반 이동 (run보다 느림). 3인칭 |
| 쪼그려 걷기 | crouch walk | 앉은 상태에서 살살 걷기. 3인칭 |
| 엎드려 이동 | pronemove | 포복 이동. 조준시 자동 해제. COD식(지향사격) vs BF식(왼팔 포복) 결정 필요 |

## 자세 전환

| 구분 | 모션명칭 |
|------|----------|
| 달리기→전력질주 | run2sprint |
| 달리기→걷기 | run2walk |
| 걷기→달리기 | walk2run |
| 걷기→전력질주 | walk2sprint |
| 전력질주→걷기 | sprint2walk |
| 앉기 | crouch (선 상태에서) |
| 앉기-조준 | crouch aim. 필요없음 |
| 서기 | Idle (앉은 상태에서) |
| 서기-조준 | stand aim. 필요없음 |
| 서기→엎드리기 | Idle2prone (조준 해제됨) |
| 앉기→엎드리기 | crouch2prone (조준 해제됨) |
| 엎드리기→서기 | prone2idle (조준 해제됨) |
| 엎드리기→앉기 | prone2crouch (조준 해제됨) |

## 점프

| 구분 | 모션명칭 | 비고 |
|------|----------|------|
| 점프 시작 | jumpstart | 비 조준 중 점프 시작~최고도~하강 시작 |
| 점프 루프 | jumploop | 비 조준 중 하락 반복 |
| 점프 착지 | jumpend | 정상 착지. 총구 아래로 크게 떨어졌다 복귀 |
| 점프 착지 피해 | jumpend_hurt | 높은 곳에서 착지. 시야가 바닥으로 훅 떨어졌다 천천히 복귀. 조준 풀림. 뼈 부러지는 소리 + 시뻘건 피해표시 |
| 점프 시작 조준 | aimjump_start | 조준 중 점프~최고도 |
| 점프 루프 조준 | aimjump_loop | 조준 중 하락 반복 |
| 점프 착지 조준 | aimjump_end | 조준 중 착지. 총기 아래로 휘청 후 조준 복귀 (조준 유지) |

## 탄창 삽탄 (샷건 전용)

| 구분 | 모션명칭 | 비고 |
|------|----------|------|
| 탄창 삽탄 시작 | reload(empty)_start | 인벤토리에서 탄종 지정 후 삽탄. 무기 집어넣고 탄입대의 탄창을 꺼내듬 |
| 탄창 삽탄 반복 | reload(empty)_loop | 한발씩 탄창에 탄환 삽탄 |
| 탄창 삽탄 완료 | reload(empty)_end | 삽탄 끝/중단. 탄창을 탄입대로 회수 |

## 총기 관찰 / 데모 모션

| 구분 | 모션명칭 | 비고 |
|------|----------|------|
| 총기 관찰1 | Inspect01 | 리시버 왼쪽면 관찰 (악세서리 확인). 셀렉터 위치 = 사격 모드 일치 |
| 총기 관찰2 | Inspect02 | 리시버 오른쪽면 비틀어 약실 관찰. 잔탄 없으면 빈 약실 보임 |
| 총기 관찰3 | Inspect03 | 탄창 빼서 잔탄 확인. 잔탄량에 따라 탄환 표시. 폴리머 탄창은 주황색 송탄판 |
| 총기 관찰4 | Inspect04 | 노리쇠 후퇴고정 후 새끼손가락으로 약실 확인. 탄환 유무에 따라 동작 차이 |
| 총기 관찰5 | Inspect05 | 총기별 개별 데모 (전반적 관찰, 권총 돌리기 등 자유 동작) |

## 기울이기 (Lean) 모션

1인칭에서는 기울이기 모션을 카메라워크로 대체 가능하나, 별도 모션 제작 시 조준 및 격발 모션과 별도 기울이기 모션 세트가 필요함.
