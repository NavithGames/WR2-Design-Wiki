캐릭터 - 사물 상호작용
■ 작동체계
● 상호작용 인터페이스
○ 상호작용 가이드라인
* 작동 가능한 사물의 상호작용 가이드 범위 내로 접근시 1차적으로 해당 물체에 테두리가 표시되어 상호작용 가능한 물체임을 표시
  

   * 상호작용 물체의 속성 중 가이드 표시 범위 (점선) 이내로 접근시 PC의 화면에 상호작용 대상 물체에 테두리를 표시함
○ 상호작용 표시
   * 상호작용 대상 물체의 상호작용 범위 내로 접근 후 시선(붉은 화살표)이 대상 물체를 향하고 있을 때 상호작용 안내가 표시되고 해당 버튼을 누르면 상호작용이 시작 됨
    
      * 상호작용 물체의 속성 중 상호작용 범위 (실선) 이내로 접근하고 시선 (→)이 대상을 바라보고 있을 때 상호작용 버튼이 화면에 표시됨
      * 모션을 지정하는 경우 대상을 향한 상호작용 준비 모션으로 변경됨
      * 상호작용 버튼을 누르면 상호작용 시작
○ 다중 상호 작용
      * 상호작용 대상에 여러 가지의 상호 작용을 할 수 있는 경우 대상에게 할 수 있는 상호작용 리스트가 버튼과 함께 표시 됨
  
         * F-G-H-J를 상호작용 표시 중 단축 키로 사용하며 최대 4종의 다중 상호작용을 벗어나지 않도록 합니다. 
         * 상호작용 안내가 나타나는 동안은 버튼의 기능은 최우선적으로 F-G-H-J에 할당 됩니다. 
         * 게임패드의 경우 상호작용 키를 길게 누르고 십자키를 눌러 상하로 상호작용 행동을 선택 후 버튼에서 손을 떼면 해당 상호작용을 실행하는 방식을 사용
■ 사물 상호작용
● 문        
         * 문을 여는 상호작용 행동
         * 문의 종류에 따른 전용 모션 사용
         * 별도의 모션이 지정되어 있지 않으면 동작을 취하지 않고 문을 열거나 닫을 수 있음
         * 문의 종류
         * 여닫이문 - 경첩 방향에 따라 밀거나 당겨서 열고 닫는 문
         * 열리는 방향이 미리 정해져 있음 
         * 미닫이문 - 슬라이드 방향으로 열리거나 닫히는 형태의 문 
         * 방향에 관계없이 열고 닫을 수 있으나 문이 벽에 막혀 있는 반대 방향에서는 상호작용이 어려움 (열 수는 있으나 닫을 수는 없음)
         * 컷씬 문 - 문과 상호작용 모션이 짤막한 컷씬 모션처럼 구성되고 상호작용 도중에는 다른 행동이 불가능한 형태의 문, 중요도가 높은 문을 여는 연출에 사용됨
● 스위치
         * 스위치를 작동시켜 스위치와 연결된 기능을 키고 끄는 작동방식
         * 문과 마찬가지로 종류에 따른 전용 모션을 사용
         * 문과 마찬가지로 모션을 지정하지 않으면 동작 없이 스위치를 키고 끔
         * 스위치의 종류
         * 버튼 스위치 - 버튼을 누를 때 마다 작동하는 스위치로 스위치 상태 변경이 없음
         * 전환 스위치 - On / Off 상태에 따라 형태가 변경되는 스위치로 상호작용 할 때 반대 상태로 형태가 변경됨
         * 컷씬 스위치 - 컷씬 문과 마찬가지로 상호작용 모션이 짤막한 컷씬 모션처럼 구성되고 상호작용 도중 다른 행동이 불가능한 형태의 스위치 
● 살펴보기
         * 문과 스위치를 제외하고 환경루팅 등 지정된 기능을 수행하기 위한 환경 상호작용 과정
         * 살펴보는 동안 지정한 형태의 자세를 유지하고 살펴보기가 끝나면 해당 자세를 해제 합니다. 
         * 살펴보기 과정
         * 살펴보기 시작
         * 전용 시작 모션을 사용하며 문, 스위치와 마찬가지로 컷씬 모션을 사용하면 시작 동작 도중 다른 행동이 차단됨
         * 살펴보기 지속
         * 살펴보는 상황이 지속되는 상황
         * 루프 모션을 이용
         * 살펴보기 대상을 들여다보는 정지형태의 모션이 주로 사용되나
         * 박스 내용물을 뒤지거나 잡동사니를 옮기는 등의 모션을 사용 할 수도 있음
         * 루팅의 경우 이 상태에서 루팅 인벤토리가 표시됨
         * 살펴보기 종료
         * 살펴보기를 끝냄
         * 마무리 모션을 사용
