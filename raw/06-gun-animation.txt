	장비
	equip
	등에 메고 있던 총기를 꺼내 지향사격 준비 자세로 전환하는 모션, 
장전 모션이 기본적으로 포함 되어 있으며 접철식 부품이 있는 무기는 해당 부품을 펼치는 모션이 포함됨
	

	해제
	unequip
	사격 준비 자세에서 무기를 집어넣는 동작, 접철식 부품이 있는 무기라도 집어넣을 때는 동작을 생략하고 신속하게 집어넣습니다.(그냥 무기를 등 뒤로 휙 제끼는 컨셉)
	

	조준
	aim
	지향사격 자세에서 조준점으로 시점을 정렬하는 모션
조준경 어태치먼트에 의해 조준점 높이가 변경되면 변경된 높이를 적용하여 블렌딩이 필요함
	RMB
	조준해제
	unaim
	조준 자세에서 지향사격 자세로 변경

	탄창제거
	release magazine
	현재 사용중인 탄창을 제거 모션 
총기 종류에 따라 탄창 제거 전 노리쇠 후퇴고정 등의 추가 모션이 필요할 수 있습니다. 
	

	탄창삽입
	insert magazine
	새로운 탄창을 삽입 모션, 탄창은 탄입대 순번에 맞는 외형의 탄창을 장착 해야 합니다. 
	

	탄창제거조준
	release magazine aim
	조준 중 탄창 제거 모션
조준선을 유지한 채로 왼손만을 이용해 탄창을 제거
	

	탄창삽입조준
	insert magazine aim
	조준 중 탄창 삽입 모션
조준선을 유지한 채로 왼손만을 이용해 탄창 삽입
	

	총기삽탄
	insert bullet
	샷건의 펌프탄창처럼 한발한발 삽탄하는 형태의 모션
	

	장전
	load
	탄창 삽입 및 삽탄이 끝난 후 장전 모션 
총기 종류에 따라 노리쇠 전진, 노리쇠 후퇴 전진, 펌프액션 등의 동작을 취함

(!) 디테일업 - 탄창이 모두 비기 전에는 별도의 장전 모션이 필요 없고 탄창을 모두 비운 뒤 탄창 교환 뒤에는 장전 모션이 필요함 
	

	장전 조준
	load aim
	조준 중 탄창 삽입 및 삽탄이 끝난 후 장전 모션 
총기 종류에 따라 노리쇠 전진, 노리쇠 후퇴 전진, 펌프액션 등의 동작을 취함
오른손 동작은 조준선 시야에 따라 보이지 않을 수도 있음

이동 
구분
	모션명칭
	비고
	키
	달리기
	run
	표준 이동 달리기, 총기는 지향사격 자세를 유지 한 채로 달리는 모션을 1인칭에서 표현 함
	

	조준달리기
	run aim
	조준 도중 표준 이동, 조준을 유지하면서 총기가 약간 흔들리는 정도로 달리기를 표현
	

	걷기
	walk
	발소리를 죽이고 천천히 걷기 1인칭 모션
	

	조준 걷기
	walk aim
	조준중 발소리를 죽이고 천천히 걷기 1인칭 모션
	

	스프린트 시작
	spring start
	전력질주, 지향사격에서 총을 늘어뜨리고 전력질주 시작
	

	스프린트 루프
	sprint loop
	전력질주 반복
	

	스프린트 끝
	sprint end
	전력질주에서 지향사격으로 복귀
	

	쪼그려 이동
	crouch run
	앉은 상태에서 일반 이동 서서 run보다 느립니다. 
	

	쪼그려 걷기
	crouch walk
	앉은 상태에서 살살 걷기
	

	엎드려 이동
	prone move
	포복 상태에서 이동, 조준시 자동으로 조준이 해제 됩니다. 
지향사격 상태로 포복 할지(COD) 왼팔로 포복 자세를 취할지 (Battlefield) 결정 필요

	prone to crouch
	엎드린 상태에서 앉기, 조준 중 앉기로 전환시 조준이 해제 됩니다. 
	

	

점프
구분
	모션명칭
	비고
	키
	점프 시작
	jump start
	비 조준 중 점프 시작에서부터 점프 최고도~하강 시작 까지 동작
	

	점프 루프
	jump loop
	비 조준 중 점프 하락 반복 동작
	

	점프 착지
	jump end
	비 조준 중 정상 착지 동작 총구가 아래로 크게 떨어졌다 기본 자세로 복귀
	

	점프 착지 피해
	jump end hurt
	착지 고도가 높은 곳에서 착지, 시야가 바닥쪽으로 훅 떨어졌다 천천히 정상 자세로 복귀됨
조준중 피해고도에서 착지시에도 조준이 풀리며 같은 모션을 사용 합니다. 
(애니 외 연출로 빠각 하는 뼈 부러지는 소리가 들리고 화면도 시뻘겋게 피해표시됨)
	

	점프 시작 조준
	jump start aim
	조준 중 점프, 조준시점에서 점프해서 최고도까지 


	

	점프 루프 조준
	jump loop aim
	조준 중 점프 최고도에서 착지 직전까지 하락 반복
	

	점프 착지 조준
	jump end aim
	조준 중 점프 착지, 총기가 아래로 휘청 하며 하강 했다 조준 상태로 복귀 
조준이 풀리지는 않음
	

	

탄창 삽탄
구분
	모션명칭
	비고
	키
	탄창 삽탄 시작
	magazine load start
	인벤토리에서 탄종을 지정 후 삽탄 했을 경우
무기를 집어넣는 동작후 선택한 탄입대의 탄창을 꺼내든다
	

	탄창 삽탄 반복
	magazine load loop
	한발 한발 탄창에 탄환을 삽탄 하는 동작
	

	탄창 삽탄 완료
	magazine load end
	탄창 삽탄이 모두 끝나거나 중도 중단 하는 경우 삽탄을 끝내고 탄창을 탄입대로 다시 회수
1인칭에서는 탄창을 화면 아래쪽으로 내리는 동작

총기 관찰 / 데모 모션
구분
	모션명칭
	비고
	키
	총기 관찰1
	gun observe 1
	총기를 지향사격 자세로부터 리시버 왼쪽면이 보이도록 들어 총몸을 관찰한다 
리시버쪽에 달린 악세서리가 잘 보이도록 
디테일업 요소 - 셀렉터가 있는 총기는 현재 셀렉터의 선택이 사격 모드와 일치하도록 함
	

	총기 관찰2
	gun observe 2
	총기를 지향사격 자세로부터 리시버 오른쪽면이 보이도록 비틀어 약실쪽을 관찰
디테일업 요소 - 총기에따라 잔탄이 없고 노리쇠 후퇴고정이 되는 경우 약실이 열리고 빈 약실이 보이도록 함
	

	총기 관찰3
	gun observe 3
	탄창을 빼 잔탄을 확인 한다. 
디테일업 요소 - 탄창에 잔탄량에 따라 탄환이 보이거나 비어있도록 한다. 
복렬 탄창의 경우 마지막 탄이 남아 있을 때는 한 쪽에 송탄판 윗면이 보이도록 한다 
폴리머 탄창의 경우 송탄판 위쪽이 밝은 주황색으로 칠해져 있다 
	

	총기 관찰3
	gun observe 4
	노리쇠 후퇴 고정 후 왼손으로 총열을 잡고 오른손 새끼손가락을 이용해 약실을 확인 한다. 
디테일업 요소 - 약실에 탄환이 있으면 약실에 삽입된 탄환이 보이고 새끼손가락이 약실 안쪽으로 들어가지 않는다
디테일업 요소 - 약실에 탄환이 없으면 약실이 비어보이고 새끼손가락이 약실 안쪽으로 들어간다. 
	

	총기 관찰4
	gun observe 5
	총기별로 개별적인 데모 동작을 취함
총을 눈앞에 들어 전반적인 모습을 관찰하거나 권총을 휙휙 돌리거나 등등
총기의 종류에 따라 자유로운 데모 동작
